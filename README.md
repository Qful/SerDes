## [简介](https://github.com/qful/USBHUB) 

通过USB外置扩展的HUB设备，具备多种主机设备适配能力，HUB集成充电宝功能，扩展实用价值，同时为多个外设提供价值。

- USB通信功能HUB
- PD模式供电转换
- 音频类设备接口
- 视频类设备接口

HUB通过单type c和键盘设备连接，接口集成视频接口和USB高速通信接口以及PD模式供电，同时设备需要维持和多PC间的USB HID外设连接，避免反复发现过程。
